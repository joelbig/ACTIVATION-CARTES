FAQ

Questions fréquentes

Quels types de cartes puis-je activer ?
Nous supportons 8 types : Transcash, PCS Mastercard, Neosurf, Steam, iTunes/Apple, Xbox, Google Play et Bitcoin. D'autres types peuvent être ajoutés sur demande.
Où trouver le code sur ma carte ?
Le code est généralement inscrit au dos de la carte, sous une zone à gratter, ou dans l'email reçu lors de l'achat en ligne. Il comporte en général 12 à 20 caractères alphanumériques.
Mon code n'est pas reconnu, que faire ?
Vérifiez que vous avez bien sélectionné le bon type de carte et que le code est saisi sans espaces. Assurez-vous que la zone argentée a bien été grattée. Si le problème persiste, contactez notre support.
L'activation est-elle immédiate ?
Oui, l'activation est traitée en moins de 5 secondes pour la plupart des types de cartes. Un email de confirmation est envoyé dès que l'activation est effective.
Mes données sont-elles protégées ?
Absolument. Toutes les communications sont chiffrées en SSL 256 bits. Nous ne stockons aucun code de carte après activation. Conformité RGPD garantie.
